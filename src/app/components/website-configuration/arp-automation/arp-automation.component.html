Generate App

Generate App option can be used to create your Android Application and you can use the same for testing or publishing on Google Play Store.

It's a simple 3 step process.

Step-1. Logo Upload

Step-2. Generate Application

Step-3. Test or Publish your Application.


Step 1:  Logo Upload

The uploaded logo will be used for the Android Application launcher icon, Welcome Screen and Login Screen of the Application.

Following properties will be required for the image you are uploading for the application. .

  1. Logo should be in PNG Format only.
  2. Logo should be a transparent image.
  3. Width of the image should be greater than 900px.
