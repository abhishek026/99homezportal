FAQ’S & Help Descriptions.

Proctur LIVE Classes


    Q. What is a host?
  1. A host is the person running a webinar, and invites participants to join.
    Q. What is a participant?
  1. A participant is a person(s) attending the live class. Participants can simply follow the link in their invitation from their desktop, phone, or other mobile device. One (1) host plan and a host can hold unlimited webinars. But If you want to run more than one webinar at a time, you will need to purchase additional licenses.
    Q. How to add Live Class?
  1. Go to Home -> Live Classes ->“+Add Live Class”
  2. Add Mandatory Fields:“Topic Name”, “Date”, Time (“From” & “To”), Teacher Name.
  3. Add Mandatory Fields:“Master Course”, “Course”, “Student”.

    Advance Setting Details:
  1. Send Notification: All the members in meeting will receive SMS notification for the session.
  2. Private Access: Only the users invited using e-mail ID would be able to join. Direct link sharing of session won't work.
  3. Access Enable Lobby: All the students will wait in waiting room after joining the session. Teachers will have access to allow students in the meeting.
  4. Hide All Notifications in recordings:When enabled this will help to hide all notifications during live class.
  5. Hide User Count:When enabled this will help to hide count of users displayed on users screen.

    Q. How to Access the Live Class-Student Portal?
  1. Student will login to account by entering valid Username and Password.
  2. Go to Dashboard > Online Classes
  3. In Upcoming Classes, Student will see the current live class schedule.
  4. Click on “Join”
  5. On Welcome Screen “Please select your Default Speaker/Headphone Option “amongst the list.
  6. Student have Joined the Live Class Successfully.
    Q. How student can access live class from Android device?
  1. Download and Install “Studde” App on your iPhone.
  2. Log in to your application.
  3. Go to >Live >Upcoming Classes > Click on “Join”
  4. Now you will have redirected to Browser (Safari and Chrome)
  5. Make sure you are using the “Desktop site” option is enabled in the chrome browser setting
    Q. How to check live class attendance?
  1. Live class attendance available only for Proctur Live not for Zoom.
  2. Attendance for each class would be available after 8:00 PM if the class ends on or before 7:30 PM.
  3. If classes end after 7:30 PM, the attendance would be available at 8:00 PM on the next day.
  4. Students will be marked present for the live class in case they attend class for the duration more than the defined threshold percentage either marked as partially present. This threshold value can be changed from setting section.
  5. If students haven't joined the class, then will be marked as absent.
  6. Notification setting can be changed from the setting section.
    Q. How to change threshold value for live class attendance?
  1. Go to Setting ->Live class notification setting-
  2. Add “Attendance Threshold (%age)”
  3. Add “Attendance Notification” if required.
  4. Click on “Save Changes”

ZOOM LIVE Classes


    Zoom Basic and Free Plan Includes:
  1. Host up to 100 participants
  2. 40-minute maximum group meetings
  3. Unlimited 1:1 Meetings

    Zoom Attendance Feature:
  1. Zoom Attendance is available for accounts with Zoom paid plans. The users which are using zoom free plans will not be getting the zoom attendance.

  1. Note: Zoom has now introduced passwords for meetings as standard, for basic plans, Admin always need to notify the password to student to join the session. Passwords can be enabled /disabled in Zoom Paid plans only.
    Q. How to integrate Zoom with Proctur?
  1. Go to “My Account” >My Account
  2. Go to “Zoom Details” & add below details. (You can add up to 2 hosts.)
    • Zoom Account Name
    • Zoom Email ID
    • API Key
    • API Secret
    • JWT Token
    • SDK Key
    • SDK Secret
    • User Limit
  3. After adding all details click on “add” and “Update”
    Q. How to student can join zoom class from Website?
  1. Student will login to account by entering valid Username and Password.
  2. Go to Dashboard > Online Classes
  3. In Upcoming Classes, Student will see the current live class schedule.
  4. Click on “Join”
  5. Student will get redirected to “Zoom launch meeting”.
    Q. How student can access Zoom live class from Android device?
  1. Download and Install “Studde or White labelled app” App on your Android device.
  2. Login to your application.
  3. Go to ->Live ->Upcoming Classes -> Click on “Join” indicated with ZOOM Label
  4. Now you will have redirected to chrome browser (Always make sure that you are using a Google chrome browser)
  5. Make sure, you are using “Desktop site” option for better performance.
    Q. How student can access Zoom live class from Android device?

  2. Download zoom application for better performance.
  3. Go to ->Live ->Upcoming Classes -> Click on “Join” indicated with ZOOM Label
  4. Now you will have redirected zoom application and class will get start.
